# DW Episode Guide

A SwiftUI App That Will Show Information On Each Doctor Who Episode With The Ability To Share The Info, Images And Links.

## Features

- macOS App
- Info On Classic Episodes
- Info On New Episodes
- Add Notes
- Share Wiki Link
- Copy Images

## Incoming Features

- iOS App
- iPadOS App
- watchOS App
- tvOS App
- Widget Support
- SJA Info
- Torchwood Info
- Class Info
- Amazon DVD Links
- Apple TV App Links
- Video Clips
- User Feedback Button
- Search To Find Episodes
- TouchBar Buttons
- Episode Air Date Notifications
